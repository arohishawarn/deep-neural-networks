# Deep Neural Network From Scratch

## Overview

In this project, I implemented a deep neural network from scratch using NumPy. The objective was to understand the mathematical and algorithmic foundations of deep learning by building the core components manually rather than relying on high-level frameworks.

The implementation supports neural networks with an arbitrary number of layers and demonstrates the complete training pipeline, including forward propagation, loss computation, backpropagation, and parameter updates.

## Concepts Implemented

* Parameter initialization
* Forward propagation
* ReLU activation
* Sigmoid activation
* Loss computation
* Backpropagation
* Gradient descent
* Multi-layer neural network architecture
* Modular neural network design

## Key Learning Outcomes

Through this project, I developed a deeper understanding of:

* How information flows through a neural network
* The role of activation functions in introducing non-linearity
* Gradient computation using backpropagation
* Parameter optimization using gradient descent
* Building scalable neural network implementations from first principles

This implementation serves as a foundation for future deep learning projects involving image classification and more advanced neural network architectures.
