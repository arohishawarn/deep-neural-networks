# Cat vs Non-Cat Image Classification Using Deep Neural Networks

## Overview

In this project, I applied the deep neural network implementation developed from scratch in the previous project to a real-world image classification task.

The objective was to classify images as either cat or non-cat using a multi-layer neural network trained on RGB image data.

## Concepts Applied

- Deep neural networks
- Forward propagation
- Backpropagation
- ReLU activation
- Sigmoid activation
- Binary image classification
- Parameter optimization using gradient descent

## Dataset

The dataset contains labeled RGB images of cats and non-cats.

Each image is represented as:

(num_px, num_px, 3)

where the three channels correspond to RGB color values.

## Results

The model was trained and evaluated on separate training and test datasets to assess classification performance.
